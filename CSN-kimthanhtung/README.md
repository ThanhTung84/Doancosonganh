# Website Học Thuật Toán Sắp Xếp

## Mô tả
Website mô phỏng trực quan các thuật toán sắp xếp dành cho sinh viên học lập trình.

## Tính năng
- Mô phỏng 5 thuật toán sắp xếp: Bubble Sort, Selection Sort, Insertion Sort, Quick Sort, Merge Sort
- Giao diện trực quan với màu sắc thay đổi theo trạng thái
- Điều chỉnh tốc độ và số lượng phần tử
- Thống kê số lần so sánh và hoán đổi
- Responsive design

## Cách chạy

### Phiên bản đơn giản (chỉ HTML/CSS/JS)
1. Mở file `index.html` trực tiếp trong trình duyệt
2. Website sẽ hoạt động ngay lập tức

### Phiên bản đầy đủ với PHP (cần web server)
1. Cài đặt XAMPP hoặc WAMP
2. Copy tất cả file vào thư mục htdocs
3. Import file `database.sql` vào MySQL
4. Cấu hình database trong `config.php`
5. Truy cập qua localhost
